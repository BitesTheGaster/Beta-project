# Beta-project

3D voxel sandbox in Godot 4.

## Status

Early development. Mechanics and design will change.

## Features

- Procedural voxel world
- Multiplayer (without anticheat)
- Block placement
- Third-person camera

## Running

1. Clone repository
2. Install Godot Voxel from [here](https://github.com/Zylann/godot_voxel)
3. Open project in Godot Voxel
4. Run it
